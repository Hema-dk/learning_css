This is a **fun project** to explore **HTML** and **CSS animations**!

## What It Does

- A **pink box (`UwU`)** bounces horizontally across the screen.  
- A **white box (`ehe`)** bounces back and forth while **rotating**.  
- Added **Font Awesome icons** that you can **click to visit my GitHub, LinkedIn, Hackerrank, and resume**.  
- Demonstrates basic CSS concepts like `@keyframes`, `flexbox`, `positioning`, and `transform`.  

## Why This Project

This project is a **playful way to learn HTML & CSS**, showing how animations and interactive icons can make a webpage dynamic and fun!  

## How to Run

1. Clone the repository:  
```bash
git clone https://github.com/Hema-dk/learning_css.git
```
2. Make sure both ```resume.html``` and ```box.css``` are in the same folder.

3. Open ```resume.html``` in your browser.

4. Click the icons or watch the boxes bounce and rotate!
